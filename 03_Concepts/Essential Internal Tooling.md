# **Essential Internal Tooling**

* **Dataview:** The foundational plugin for treating Markdown files as a relational database. It allows users to write SQL-like queries to generate dynamic tables, lists, and task views based entirely on YAML frontmatter, providing real-time visibility into the vault's structure.42  
* **Smart Connections:** A local-first artificial intelligence plugin that embeds the vault locally and surfaces semantically related notes in real-time. It operates as a localized Retrieval-Augmented Generation system, providing the benefits of vector search without exposing sensitive data to external cloud application programming interfaces.53  
* **Model Context Protocol Integrations:** Plugins such as Semantic MCP enable external agents like Claude Desktop or Claude Code to securely access the vault as a file system, query Dataview directly, and execute complex graph traversals.24

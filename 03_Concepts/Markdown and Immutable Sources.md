# Markdown and Immutable Sources

In a world of shifting platforms and walled gardens, raw text remains the universal anchor. Markdown serves as the optimal medium for this because it is inherently machine-readable while remaining human-legible. By treating the original document as an immutable source, we preserve the integrity of the data while allowing Python scripts to tear down and rebuild the information into highly connected, atomic structures.

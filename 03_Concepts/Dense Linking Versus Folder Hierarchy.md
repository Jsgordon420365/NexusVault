# **Dense Linking Versus Folder Hierarchy**

For human recall, folder hierarchies provide a comforting sense of spatial orientation and strict categorization. For machine readability, dense bidirectional linking is vastly superior. A vector retrieval system parsing a Zettelkasten will pull an atomic note and immediately perceive its semantic neighborhood via its explicit backlinks.14 Conversely, a model retrieving a document from a deep PARA folder structure lacks the relational context unless the preprocessing script explicitly appends the directory path to the text chunk. Therefore, an artificial intelligence-optimized vault heavily favors a flat, deeply linked architecture augmented by queried Maps of Content, utilizing folders strictly for gross categorization such as separating raw ingestion sources from synthesized output.5

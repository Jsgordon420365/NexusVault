# **The Link Graph and Semantic Navigation**

Obsidian elevates the hyperlink to a structural primitive. Using wiki-link syntax, users establish a densely connected graph of concepts. This bidirectional linking automatically generates backlinks, allowing a human user—and critically, an artificial intelligence agent—to traverse from a central concept out to every instance where that concept was referenced.2 The visualization of this network via the Graph View provides an immediate diagnostic of vault topology, revealing orphaned notes, dense clusters, and emergent semantic hubs.4 Canvases further extend this spatial reasoning by allowing users to map Markdown files visually, creating infinite spatial whiteboards that preserve the underlying file structure.3

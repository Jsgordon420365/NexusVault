# **Machine-Assisted Diagnostics**

While models should not execute structural changes autonomously, they serve as exceptional diagnostic tools. Utilizing the Model Context Protocol, a frontier model can act as a Vault Inspector. The agent can be prompted to run a comprehensive health check: scanning the directory for notes missing critical metadata, identifying broken or unresolved wiki-links, and surfacing near-duplicate concept notes for potential merging.50

Crucially, the maintenance workflow must preserve trust and reversibility. The model must only propose changes or output a diagnostic report. Granting an autonomous agent silent read and write permissions to independently refactor a vault's structure inevitably leads to catastrophic data loss, recursive hallucinations, and the irreversible destruction of human-curated semantic links.51
